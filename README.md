## Stack

- ReactJS
- Parcel v2
- React-Swipeable


## Development

Want to contribute? Great!

Chronique de l'attente uses Parcel for fast developing.

First:

```sh
yarn install
```

Then:

```sh
yarn start 
# or yarn dev
```
